Este repositorio contiene los ejercicios y materiales para la actualización del curso curso **[Desarrollo de Apis en Java con Spring Boot](https://todocodeacademy.com/course/desarrollo-de-apis-con-spring-boot/)** ofrecido por la **[TodoCode Academy](https://todocodeacademy.com/)**.

## ¿Cómo usar este repositorio?

Cada carpeta en contiene un ejercicio diferente o código implementado por clase. Se recomienda que sigas el orden de los ejercicios en base a lo que veas en el video explicativo del curso, ya que los mismos están diseñados para aumentar en dificultad progresivamente. 
En caso de que se trata de un ejercicio, se sugiere intentar resolver el mismo por tu cuenta antes de revisar las soluciones que aquí se presentan para seguir ejercitando tu lógica de programación.
